Progress block
=
Блок, отображающий прогресс в виде круга.

Demo: https://chibik52.github.io/ProgressBlock/

Установка
-

Скачайте progress.js и добавьте скрипт в html:<br>
```<script src="progress.js"></script>```

Подключение дополнительных css файлов не требуется

Состояния
-
Normal – базовое состояние, в котором путём задания Value можно управлять
размером дуги, отражающей некоторый параметр или прогресс процесс. Начало
дуги соответствует 12 часам, конец дуги при увеличении параметра Value
движется по направлению часовой стрелки и достигает начала при значении 100.<br>
Animated – независимое состояние, при котором блок 
начинает вращаться с некоторым периодом по часовой стрелке.<br>
Hidden – состояние скрывающее блок со страницы.

Использование
-
В разметке HTML добавьте id к элементу, внутрь которого нужно поместить блок прогресса.
<br>
Например:<br>
```<div id=block></div>```

Далее в вашем js коде создайте экземпляр класса ```ProgressBarBlock```
, где необходимо передать обязательный параметр container - id элемента, внутрь которого нужно поместить блок прогресса. При необходимости возможно поменять значения по умолчанию, передав их во втором параметре в виде объекта со следующими ключами:<br>
|Ключ|Тип|По умолчанию|Описание|
|-|-|-|-|
|`value`| Number | 0 |Начальное значение прогресса блока|
|`size`| Number | 200 |Размер блока|
|`strokeWidth`| Number | 20 | Толщина линии круга |
|`backgroundColor`|String| #eef3f6 | Цвет линии фонового круга |
|`progressColor`| String | #005dff | Цвет линии круга прогресса |
|`animated`| Boolean | false | Анимированный блок |
|`hidden`| Boolean | false | Скрытый блок |

Создание экземпляра с параметрами по умолчанию:<br>
```
const progressBlock = new ProgressBarBlock("block");
```
Создание экземпляра со своими параметрами:<br>
```
const progressBlock = new ProgressBarBlock("block", {size: 130, strokeWidth: 15, progressColor: "#857dff"});
```
<br>

API:
-
После создания экземпляра класса `ProgressBarBlock` например, `const progressBlock = new ProgressBarBlock(...);` доступны следующие свойства и методы:

|Метод/Свойство|Описание|Пример|
|-|-|-|
|`value`|Установить значение прогресса (от 0 до 100)|`progressBlock.value = 50;`|
|`value`|Получить значение прогресса (от 0 до 100)|`let currentValue = progressBlock.value;`|
|`setAnimated`|Включить/отключить анимацию блока (true/false)|`progressBlock.setAnimated(true);`|
|`setHidden`|Скрыть/показать блок (true/false)|`progressBlock.setHidden(true);`|
